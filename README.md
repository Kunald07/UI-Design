# UI DESIGN

> Visual design that turns ideas into real-world interfaces.

Modern visual design playground built to help designers and developers create clean, scalable, and production-ready interfaces faster. Inspired by modern design tools, Rachna focuses on simplicity, performance, and an intuitive user experience.

## ✨ Features

### 🎨 Infinite Canvas

Design without boundaries using an infinite canvas that supports smooth panning and zooming.

### ⚡ Command Palette

Access actions instantly with a Raycast-inspired command palette using `Ctrl + K`.

### 💾 Local Storage

Your work is automatically saved locally, ensuring designs persist across sessions.

### ⌨️ Keyboard Shortcuts

Speed up your workflow with familiar design-tool shortcuts.

### 🚀 HTML & CSS Export

Generate clean, production-ready code that developers can use immediately.

### 🖥️ Minimal & Modern Interface

A distraction-free workspace focused on helping you build great user experiences.
